# 🍔 Kaju Bhai Food Dhaba - Food Ordering Website

A modern and responsive food ordering website UI built using HTML, CSS, JavaScript, Swiper.js, and GSAP animations.  
This project simulates a restaurant/food delivery platform with login system, menu sections, sliders, and smooth animations.

---

## 🚀 Features

-  Beautiful landing page UI
-  Login popup with GSAP animation
-  Customer testimonial slider (Swiper.js)
-  Interactive food menu section
-  "Order Now" buttons with interaction
-  Fully responsive design
-  Smooth animations using GSAP
-  Navigation bar with sections
-  Contact section with details
-  Custom cursor animation (optional JS)

---

## 🧑‍💻 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- GSAP (GreenSock Animation Platform)
- Swiper.js
- Font Awesome Icons
